# Дипломное приложение // DevOps

Веб-приложение со встроенным виджетом текущего времени в стилистике художественной вселенной Marvel. Проект подготовлен в рамках дипломной работы по курсу «DevOps-инженер».

## 🛠 Технологический стек
* **Frontend:** HTML5, CSS3 (Custom Properties, Анимации), JavaScript (Vanilla JS)
* **Веб-сервер:** Nginx
* **Контейнеризация:** Docker
* **CI/CD:** GitHub Actions

---

## 📁 Структура проекта
* `.github/workflows/ci-cd.yml` — конфигурация автоматического пайплайна сборки и деплоя.
* `src/index.html` — исходный код веб-страницы (интерфейс, стили и логика часов).
* `.gitignore` — список исключений для Git.
* `Dockerfile` — инструкция для сборки Docker-образа на базе Nginx.
* `nginx.conf` — конфигурационный файл веб-сервера.
